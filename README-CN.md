rosedb 是一个稳定、快速、内嵌的 NoSQL 数据库，支持多种数据结构，包含 `String`、`List`、`Hash`、`Set`、`Sorted Set`。

rosedb 类似于 `Redis`，但是可以选择将 value 存储到磁盘中。

主要特性：

* **支持丰富的数据结构，包含 `String`、`List`、`Hash`、`Set`、`Sorted Set`**
* **内嵌使用，维护成本低**
* **高性能，对读写负载的应用都适合**
* **数据不局限于内存容量**
